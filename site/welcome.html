Welcome to the CMS

Posted by Administrator

Welcome to the test CMS. This CMS provides a lot of different features, including a default dark mode, animated transitions for posts, and beautiful material design components. A lot of unique features are available when using Material Design Components for the Web. The project is officially supported by Google, and adheres to all their design standards. More and more components are regularly added, with support also available via their Discord server and GitHub page. They also have components available for iOS and Android, as well as their own framework named "Flutter".
Besides being able to use material design in every conceivable platform, Material.io also provides guidelines for how to use Material Design in the best way, especially in terms of user experience.
